# Context Directory

This directory contains company and client information for research customization.

## Files

- `company-profile.md.example` - Template for company profile
- `client-info.md.example` - Template for client information
- `industry-background.md.example` - Template for industry context

## Usage

Copy example files and remove `.example` extension:

```bash
cp company-profile.md.example company-profile.md
cp client-info.md.example client-info.md
cp industry-background.md.example industry-background.md
```

Then edit files with your specific information.
